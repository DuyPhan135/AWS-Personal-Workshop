#### Add Custom Domain for Amplify Frontend.
Access Amplify, select Add custom domain.
![custom-fe](/images/5-Workshop/5.5-DeployFE/5.5.4-Custom-Domain-FE/custom-fe.png)

Continue to select Add domain.
![custom-fe1](/images/5-Workshop/5.5-DeployFE/5.5.4-Custom-Domain-FE/custom-fe1.png)

Select the domain added to the Hosted Zone in Route53 then select Configure domain.
![custom-fe2](/images/5-Workshop/5.5-DeployFE/5.5.4-Custom-Domain-FE/custom-fe2.png)

Check Set up redirect v.v to ensure that any user entering www.webchat.mom will automatically be redirected to webchat.mom, helping to provide a consistent and optimal experience. Then click Add domain.

![custom-fe3](/images/5-Workshop/5.5-DeployFE/5.5.4-Custom-Domain-FE/custom-fe3.png)

Wait a few minutes for the Custom Domain to be ready.
![custom-fe4](/images/5-Workshop/5.5-DeployFE/5.5.4-Custom-Domain-FE/custom-fe4.png)

And we are done.
![custom-fe5](/images/5-Workshop/5.5-DeployFE/5.5.4-Custom-Domain-FE/custom-fe5.png)

Enter webchat.mom in the browser to view the result.
![custom-fe6](/images/5-Workshop/5.5-DeployFE/5.5.4-Custom-Domain-FE/custom-fe6.png)
